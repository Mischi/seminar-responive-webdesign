Mithilfe von CSS Transformationen können (Html-) Elemente in einem zwei und drei dimensionalen Koordinatensystem verschoben, gedreht, skaliert und geschert werden. 

width: 200px;
height: 200px;
background-color: #FF6600;
						
transform: scale(0.5);
						
	transform: rotate(45deg);
							
	transform: rotate(45deg);

Mithilfe von CSS Transitions können Änderungen von CSS Eigenschaften über eine definierte Zeitspanne von Zustand A nach Zustand B erfolgen



transition: all 10s ease-in-out;
transform: translateX(800px) rotate(135deg); 
			


transition: background-color 5s linear;
background-color: blue;

Mittels CSS Animations lassen sich wie mit Transitions, Animation erstellen. Diese basieren jedoch auf Keyframes und sind explizit!



@keyframes rotateTo {
  to {
    transform: translateX(800px) 
    		   rotate(360deg); 
  }
}
					
#elem {
  animation-name: rotateTo;
  animation-duration: 5s;
  animation-iteration-count: infinite;
  animation-direction: alternate;
  animation-timing-function: linear;
}
					


@keyframes rotateToRectMove {
  20% { transform: translateY(100px) rotate(90deg); }
  80% { transform: translateX(700px) translateY(100px) rotate(270deg); }
  100% {transform: translateX(700px) rotate(360deg); }
}
